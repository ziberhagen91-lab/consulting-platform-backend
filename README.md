# Consulting Platform

A modern full-stack CRM and task management platform built with **Next.js**, **NestJS**, **Prisma**, and **PostgreSQL**.

The application helps consultants and agencies manage clients, organize tasks, and monitor business activity through a clean and responsive interface.

## 🚀 Features

* 🔐 JWT Authentication (Login & Registration)
* 👥 Client Management (Create, Read, Update, Delete)
* 📋 Task Management (Create, Read, Update, Delete)
* 🔗 Link tasks to specific clients
* 📊 Dashboard with analytics
* 🌍 Bilingual interface (Ukrainian / English)
* 🔍 Search and filtering
* 📱 Responsive design
* 🗄️ PostgreSQL database with Prisma ORM

## 🛠️ Tech Stack

### Frontend

* Next.js
* React
* TypeScript
* Tailwind CSS

### Backend

* NestJS
* Prisma ORM
* PostgreSQL
* JWT Authentication

## 📂 Project Structure

* Frontend: Next.js application
* Backend: NestJS REST API
* Database: PostgreSQL managed with Prisma

## ⚡ Installation

### Backend

```bash
npm install
npm run start:dev
```

### Frontend

```bash
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file for the backend with values similar to:

```env
DATABASE_URL=your_postgresql_connection_string
JWT_SECRET=your_secret_key
```
